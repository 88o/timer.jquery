jQuery Timer

Start/Stop/Resume a timer inside any HTML element.



Demo


Usage

Start: $("#divId").timer("start") OR $("#divId").timer({action:"start", seconds: 3400})
Pause: $("#divId").timer("pause")
Resume: $("#divId").timer("resume")
Destroy: $("#divId").timer("destroy")
Get number of seconds: $("#divId").data("seconds")
