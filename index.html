jQuery Timer Plugin

A simple timer plugin that makes use of the jQuery library.

Usage

The plugin currently has 3 methods which can be called on the DIV element or any other HTML element that you wish to display a timer in. These methods are:

  • Start: Initializes the timer on the HTML element. Example: $("#divId").timer("start"). You can even start the timer from a particular time by specifying seconds. $("#divId").timer("start", 3400)
  • Pause: Pauses an existing timer. Example: $("#divId").timer("pause")
  • Resume: Resumes a paused timer. Example: $("#divId").timer("resume")
